1. O que é o Node.js? R: é um ambiente de execução JavaScript de código aberto e multiplataforma que permite executar códigos JavaScript no lado do servidor (backend), fora do navegador.

2. O Node.js usa qual linguagem de programação? R: JavaScript.

3. O Node.js roda no navegador ou fora dele? R: Fora do navegador.

4. Para que serve o Node.js? R: Executar JavaScript fora do navegador.

5. Qual comando é usado para executar um arquivo Node.js no terminal? R: node <nome_do_arquivo.js>

6. O que é o npm? R: Node Package Manager é o gerenciador de pacotes padrão para Node.js

7. O que significa a palavra módulo no Node.js? R: São aqueles pacotes criados por outros desenvolvedores (e empresas) que detém a habilidade de criar módulos, o que não é uma tarefa muito dificil. Esses módulos podem ser facilmente integrados ao seu projeto usando o npm (Node Package Manager).

8. O Node.js é mais usado no front-end ou no back-end? R: é utilizado predominantemente no back-end.

9. O que é o arquivo package.json? R: é o arquivo de configuração central de projetos Node.js e JavaScript

10. O Node.js é gratuito ou pago? R: é totalmente gratuito
